# Research Docs

Research documentation, reports, and notes.

## Structure

```
rng/            - Random circuit pseudorandomness testing (dieharder)
  RNG_REPORT.md   - Main report: methodology, results, analysis
  RNG_TEST_PLAN.md - Sweep configurations and acceptance criteria

infra/          - Infrastructure and environment docs
  API_SCAN.md       - API endpoints inventory
  CIRCUIT_FORMATS.md - Circuit representation formats
  DB_INVENTORY.md   - Database catalog
  DB_SCHEMA_MAP.md  - Schema documentation
  DEPLOYMENT.md     - Deployment notes
  ENVIRONMENT.md    - Dev environment setup

SWAP_FLIP_GADGETS.md - Swap/flip gate gadget constructions
```

## Projects

- **Local Mixing** - Random reversible circuits, gate models, mixing analysis
- **RNG Testing** - Statistical testing of circuit-generated bitstreams (dieharder)
- **Identity Factory** - Circuit identity search and database
- **SAT Revsynth** - SAT-based reversible synthesis
- **Obfuscated Circuits** - Circuit obfuscation metrics
