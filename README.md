# Research Docs

Research documentation, reports, and notes for the research group projects.

## Structure

```
rng/          - Random circuit pseudorandomness testing (dieharder)
```

## Projects covered

- **Local Mixing** - Random reversible circuits, gate models, mixing analysis
- **RNG Testing** - Statistical testing of circuit-generated bitstreams
- **Identity Factory** - Circuit identity search and database
- **SAT Revsynth** - SAT-based reversible synthesis
